# PostHub – Content Management Application

A full-stack content management system that allows users to create, edit, and manage posts dynamically.

## Features
- Create new posts
- Edit existing posts
- Delete posts
- Dynamic rendering using EJS
- RESTful routing

## Tech Stack
- Node.js
- Express.js
- EJS
- CSS

## How to Run
1. npm install
2. node index.js
